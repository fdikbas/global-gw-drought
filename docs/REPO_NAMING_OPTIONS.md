# Suggested repository names

## Recommended
- `global-gw-drought-paper1`

Reason:
- clear
- short enough for GitHub
- explicitly distinguishes Paper 1 from the planned second paper

## Good alternatives
- `global-gw-drought-screening`
- `global-groundwater-anomaly-screening`
- `gw-depletion-screening-paper1`

## Not recommended
- names that imply the full project is public if Paper 2 material is intentionally excluded
- names tied only to one figure block or one script version
