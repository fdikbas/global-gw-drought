# Repository scope for Paper 1

This repository is intentionally scoped to the manuscript:

**Identifying persistently depleting aquifers from global groundwater anomalies**

Included:
- anomaly construction
- HybridQC
- decadal aggregation
- Paper 1 post-analysis summaries

Excluded:
- standalone figure-polishing scripts
- manuscript-only cosmetic plotting helpers
- material reserved for the planned second paper
- unrelated exploratory outputs

This scope control helps keep the public repository aligned with the claims and reproducibility requirements of Paper 1.
